# Pi-Xcels Movie Fullstack Application

This is a simple fullstack web application built using React and Express. 
here's the working link - https://pi-xcels-assasment.onrender.com/

It displays a list of movies and allows users to view detailed information about each movie.

## Features

- List all movies
- View single movie details
- Responsive layout (grid on desktop, single column on mobile)
- Backend API built with Express
- Data loaded from local JSON file

## Tech Stack

Frontend:
- React
- React Router
- CSS

Backend:
- Node.js
- Express

## API Endpoints

GET /api/movies  
Returns list of movies

GET /api/movies/:id  
Returns details of a single movie
